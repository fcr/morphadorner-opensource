Syllable counter.

A syllable counter counts the number of syllables in a word (spelling).

All MorphAdorner syllable counters must implement the {@link edu.northwestern.at.utils.corpuslinguistics.syllablecounter.SyllableCounter} interface. The {@link edu.northwestern.at.utils.corpuslinguistics.syllablecounter.SyllableCounterFactory} provides the mechanism for instantiating a default or specified instance of a syllable counter implementation.

MorphAdorner provides an English language syllable counter as the default
