Thesaurus.

A Thesaurus provides synonyms (word related in meaning) and antonyms (words opposite in meaning) to a specified word.

All MorphAdorner thesauruses must implement the {@link edu.northwestern.at.utils.corpuslinguistics.thesaurus.Thesaurus} interface. The {@link edu.northwestern.at.utils.corpuslinguistics.thesaurus.ThesaurusFactory} provides the mechanism for instantiating a default or specified instance of a thesaurus implementation.
